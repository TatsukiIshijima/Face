# Face
Pupil detect used by dlib

## Demo  
**Preparation**  
Download shape_predictor_68_face_landmarks.dat from dlib(http://dlib.net/files/dlib-19.3.zip)  

**Run**  
python face.py image
image is imagefile reflected someone face.

## Requirement  
1. Python3(Anaconda)  
2. Numpy(Anaconda)  
3. matplot(Anaconda)  
4. dlib  
4. OpenCV3 or OpenCV2
